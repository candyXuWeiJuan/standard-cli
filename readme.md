### es-dev-standard 前端开发规范

#### 使用方法

1. 安装

```bash
	$ npm i standard-cli -g
```

2. 使用
    - 初始化所有规则
    ```bash
    $ standard -i
    ```
    - 仅初始化 eslint 规则
    ```bash
    $ standard -e
    ```
    - 仅初始化 prettier 规则
    ```bash
    $ standard -p
    ```
    - 仅初始化 git-message 规则
    ```bash
    $ standard -h
    ```
